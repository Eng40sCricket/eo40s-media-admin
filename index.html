GitHub Token
A fine-grained Personal Access Token scoped to the eng40s-website repository. Required for all uploads and file edits.
Create a fine-grained token at github.com → Settings → Developer Settings → Personal Access Tokens. Scope it to Eng40sCricket/eng40s-website only, with Contents: Read & Write permission. The token is stored in your browser only.
